发行人注册于英属维京群岛的有限公司Himalaya International Reserves Limited，是以以太坊(Ethereum)区块链技术为基础的ERC-1404代币，与ERC-20协议标准具有后兼容性。HDO是稳定币，HDO与美元挂钩，由包括美元和现金等价资产在内的储备金支持。HDO通过与美元维持固定价值，为交易人和投资者提供了一个安全的避风港。HCN和HDO可互相兑换。 

区块链的不可逆转性需要消耗庞大的计算资源用于验证，公链的交易时间非常缓慢。Himalaya生态系统采用以太坊(Ethereum)和Quorum区块链的混合链。Quorum是由摩根大通以企业为中心开发的基于以太坊的开源分类帐协议。
